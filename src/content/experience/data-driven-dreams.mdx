### Data Export Example
Users can export their data in JSON format.

```json
{
  "userId": "user-123",
  "sleepData": [
    {
      "date": "2023-01-09",
      "duration": "8h 15m",
      "quality": "good"
    }
  ],
  "dreamJournal": [
    {
      "date": "2023-01-09",
      "entry": "Dreamt of flying over mountains.",
      "sentiment": "positive"
    }
  ]
}
```
